## ⌨️Installation

1. Go to `Extensions`
2. Search for `Lotus Theme` and click `Install`
3. Select `Color Theme` and choose `Lotus Dark` or `Lotus Light`

> Font in screenshots: [Cascadia Code](https://github.com/microsoft/cascadia-code)
